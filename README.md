# Customer-Churn-Prediction
This project predicts customer churn using machine learning techniques.
## Dataset
IBM Telco Customer Churn Dataset

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- Recall
- ROC-AUC

## Technologies
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn
- XGBoost

## Results
Random Forest achieved the best performance with high ROC-AUC score.
